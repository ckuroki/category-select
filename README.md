## Category Select react component
---

React dropdown select component with parent-child options support.

### Installation

Clone repository
```bash
  git clone https://github.com/ckuroki/category-select.git 
```

Install dependencies
```bash
  $ cd category-select
  $ npm install
```

Run development server
```bash
  $ npm start
```

Check component code on `src/CategSelect.js`.

or 

See complete article on [kombi.io](http://kombi.io/category-select/)
